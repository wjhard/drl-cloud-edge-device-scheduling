# Repository Audit Evidence

This directory contains reproducibility checks generated during the final repository audit:

- `technical_report_result_audit.log`: report path, numeric-result, evidence, Git tracking, and privacy checks.
- `pytest_full_output.log`: complete project test-suite output.
- `checkpoint_load_check.log`: final residual checkpoint load verification and SHA-256 digest.

All paths in these logs are repository-relative or anonymized.
